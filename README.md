> O app definitivo para organizar seus estudos e matérias. 

Organize suas matérias, horários e compromissos acadêmicos em um só lugar. Feito para estudantes que precisam de ordem no caos.

---

## ✨ Funcionalidades

* **📅 Gestão de Matérias:** Adicione e organize suas disciplinas.
* **💾 Backup JSON:** Exporte e importe seus dados para nunca perder nada.
* **🚀 Performance:** Desenvolvido com Flutter para uma experiência fluida.
* **🌙 Interface Intuitiva:** Focada no que importa: seu estudo.
